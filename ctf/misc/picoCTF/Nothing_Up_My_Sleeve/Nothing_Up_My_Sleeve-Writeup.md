# Nothing Up My Sleeve Writeup

## 1. 問題概要

本問は接続確認を目的とした基礎的なフォレンジック系問題であり、追加の解析や逆アセンブルを必要としない。チャレンジインスタンス内に平文のフラグがそのまま配置されており、ユーザは `flag.txt` を参照するだけでよい。問題説明にある「in-the-clear」は、暗号化や難読化が行われていないことを意味する。

## 2. 作業内容

* チャレンジインスタンスにログインまたはアクセスする。
* 作業ディレクトリに `flag.txt` が存在することを確認する。
* そのままファイル内容を表示する。

## 3. 取得したフラグ

```
picoCTF{c0ngr4ts_0n_y0ur_s4n1ty}
```

## 4. 解法のポイント

* 特別な調査は不要であり、ファイル参照のみで完結する。
* 通常の CTF における接続テスト・環境確認用の問題に該当する。

## 5. 注意点

* 実際のフォレンジック問題ではログ解析・パケット解析・ファイル復元などが必要になるが、本問ではそれらの工程は存在しない。
* 難読化・圧縮・暗号化が行われていないため、探索範囲は極めて限定される。
