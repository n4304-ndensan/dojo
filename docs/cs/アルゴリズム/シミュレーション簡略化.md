# シミュレーション簡略化（Simulation Simplification）

## 概要

複雑なシミュレーションをすべて逐次的に実行せず、**状態遷移や規則を抽象化・圧縮して高速に結果を得る手法**。問題の本質的な変化部分のみを抽出することで、計算量を削減する。

## 目的

* 全探索や逐次シミュレーションが $O(N^2)$ 以上になる場合に、不要な中間状態を省略。
* 制約を満たす範囲で**数式化・状態圧縮**を行う。

## 手法の例

* **状態圧縮**：数値・フラグ・区間情報などを配列・ビット列で管理。
* **変化イベントのみ処理**：時間変化を全ステップで追わず、変化が起こる点だけを計算。
* **周期性利用**：一定周期で同じ状態に戻る場合、周期を検出してスキップ。

## 例

例：粒子が左右に動くシミュレーション

```
逐次法：毎ステップで全粒子更新 → O(N*T)
簡略法：速度が一定 → 位置 = 初期 + 速度×T で一括計算 → O(N)
```

## 注意点

* 簡略化しすぎると誤差・抜け漏れが生じるため、境界条件の扱いに注意。
* 厳密性が必要な場合（数値解析や物理再現）は原法シミュレーションとの検証を行う。
