# 回文生成（Palindrome Generation）

## 概要

**回文生成**とは、与えられた文字列・数字・集合などから**左右対称な文字列（回文）**を作る手法。文字列処理や列挙問題で多用される。

## 方針

### 1. 中心展開法（Center Expansion）

各位置を中心にして左右に広げることで回文を構築。

```python
def expand(s, l, r):
    res = []
    while l >= 0 and r < len(s) and s[l] == s[r]:
        res.append(s[l:r+1])
        l -= 1; r += 1
    return res
```

### 2. 組み合わせ生成

- 文字の出現回数から、左右対称な並びを構築。
    
- 奇数回出現する文字は中央に1つのみ配置可能。
    

```python
from collections import Counter
cnt = Counter("aabbcc")
# 例: abccba
```

### 3. DPを用いた最長回文部分列（LPS）

```python
dp[i][j] = (s[i]==s[j] and dp[i+1][j-1])
```

## 応用

- 最長回文部分列 / 部分文字列
    
- 回文判定フィルター（例：数値・辞書単語）
    
- 文字列生成コンテスト問題（AtCoder等）
    

## 注意

- 回文生成の総数は指数的に増えるため、列挙時は制約を付けること。
    
- Unicodeを扱う場合、正規化（NFC/NFKC）に注意。
