# Azure AD B2C を利用した外部ユーザー認証

## 1. 背景（シナリオ）

ある組織では、外部ユーザーが利用するWebアプリケーションを開発しています。このアプリケーションでは、ユーザーが既存のアカウントを使用してログインできるようにする必要があります。

具体的には、Google、Facebook、Microsoftなどの外部IDプロバイダーを利用したログイン機能を提供することが求められています。さらに、このアプリケーションにはユーザー自身がアカウント管理を行えるセルフサービス機能も必要です。

ユーザーは自分のプロフィールを編集でき、パスワードを忘れた場合にはセルフサービスでパスワードをリセットできるようにする必要があります。このような機能を安全かつスケーラブルに提供できるID管理サービスが必要になります。

## 2. 要件整理

このシナリオの要件を整理すると、次のようになります。

このシステムでは、外部ユーザーの認証とユーザー管理機能を統合して提供する必要があります。

- 外部ユーザー向けの認証機能  
- Google、Facebook、Microsoftなどのソーシャルログイン対応  
- ユーザー自身によるアカウント管理  
- セルフサービスによるパスワードリセット  
- プロフィール編集機能  
- 安全でスケーラブルなID管理  

これらの要件から、消費者向けのID管理サービスを利用する必要があることが分かります。

## 3. 技術の基本概念

Azure Active Directory B2C（現在は Microsoft Entra External ID for customers と呼ばれる場合もあります）は、消費者向けアプリケーションの認証とユーザー管理を提供するクラウドIDサービスです。

Azure AD B2C は、企業内部ユーザーを管理する Azure Active Directory（Microsoft Entra ID）とは異なり、外部ユーザーや顧客向けのID管理に特化しています。

このサービスでは、ユーザーがソーシャルアカウントを利用してログインすることができます。また、ユーザー自身がアカウントを管理できるセルフサービス機能も提供されます。

## 4. アーキテクチャまたは設計のポイント

Azure AD B2C を導入すると、アプリケーションの認証機能を外部IDプロバイダーと統合することができます。

Azure AD B2C は Google、Facebook、Microsoft などのソーシャルIDプロバイダーと連携でき、ユーザーは既存のアカウントを使用して簡単にログインできます。

また、Azure AD B2C ではユーザーフローと呼ばれる仕組みを利用して、次のような操作を提供できます。

- ユーザー登録  
- サインイン  
- パスワードリセット  
- プロフィール編集  

これにより、ユーザーは管理者の介入なしでアカウントを管理できるようになります。

## 5. 設計判断（なぜこの構成になるか）

この問題では「外部ユーザーの認証」「ソーシャルログイン」「セルフサービスユーザー管理」という3つの要件が重要です。

Azure AD B2C は、これらの機能をすべて提供する消費者向けID管理サービスです。

さらに、Azure AD B2C は次のような特徴を持っています。

- ソーシャルログインの統合  
- カスタマイズ可能なユーザーフロー  
- セルフサービスパスワードリセット  
- 高いスケーラビリティ  

そのため、外部ユーザー認証を管理する最適なAzureサービスとなります。

## 6. 他の選択肢が誤りな理由

B. Azure Active Directory  

Azure Active Directory（Microsoft Entra ID）は主に企業内部ユーザーや組織の従業員向けのID管理サービスです。消費者向けアプリケーションには適していません。

C. Azure API Management  

API Management はAPIの公開や管理を行うサービスであり、ユーザー認証やユーザー管理機能を提供するサービスではありません。

D. Azure App Service Authentication  

App Service Authentication は簡易的な認証機能を提供しますが、ユーザー管理やソーシャルログイン、セルフサービスアカウント管理などの高度なID管理機能は提供しません。

## 7. 最終回答

A. Azure Active Directory B2C

## 8. まとめ

消費者向けアプリケーションでは、外部ユーザーの認証とユーザー管理を安全に提供することが重要です。

Azure AD B2C は次の機能を提供します。

- Google / Facebook / Microsoft などのソーシャルログイン  
- セルフサービスパスワードリセット  
- プロフィール編集  
- スケーラブルなユーザー管理  

そのため、外部ユーザー認証とユーザー管理を提供する最適なAzureサービスは Azure Active Directory B2C です。
