# Azure VNet 間通信（グローバル VNet ピアリング）

## 1. 背景（シナリオ）

ある企業では、**異なる Azure リージョンに 2 つの Virtual Network（VNet）** を展開しています。  
これらの VNet 内のリソース同士が通信できるようにする必要があります。

この通信には次のような要件があります。

安全な通信であること  
最小限の遅延であること  
異なるリージョン間で接続すること  

つまり、**リージョン間の安全で低遅延なネットワーク接続**を構築する必要があります。

この要件を満たす Azure ネットワーク構成を選択する必要があります。

## 2. 要件整理

問題文から重要な要件を整理すると次の通りです。

まず、2つの VNet は異なる Azure リージョンに存在しています。

次に、VNet 内のリソース同士が直接通信できる必要があります。

さらに、通信は安全であり、パフォーマンス要件として **低遅延通信**が求められています。

これらの要件をまとめると次のようになります。

リージョン間接続  
安全な通信  
低遅延通信  
VNet 間の直接通信  

この条件を最も満たす Azure 機能が **Global VNet Peering** です。

## 3. 技術の基本概念

Azure Virtual Network Peering は、2つの VNet を直接接続する機能です。

ピアリングされた VNet は、同じネットワークに存在するかのように通信できます。

VNet Peering の特徴は次の通りです。

まず、通信は **Azure バックボーンネットワーク**を通ります。

次に、通信は **パブリックインターネットを通過しません**。

さらに、ネットワークゲートウェイを必要とせず **低遅延通信**を実現できます。

異なるリージョンに存在する VNet を接続する場合、この機能は **Global VNet Peering** と呼ばれます。

## 4. アーキテクチャまたは設計のポイント

このシナリオでは、2つのリージョンにある VNet を **Global VNet Peering** で接続します。

この構成では、Azure のバックボーンネットワークを使用して VNet 間の通信が行われます。

この設計には次の特徴があります。

パブリックインターネットを通らない通信  
低遅延のネットワーク通信  
安全なプライベート接続  
ゲートウェイ不要のシンプル構成  

このため、アプリケーション間通信やマイクロサービス間通信に最適です。

## 5. 設計判断（なぜこの構成になるか）

Global VNet Peering は Azure の内部ネットワークを利用するため、非常に低い遅延で通信できます。

また、VPN ゲートウェイのように暗号化トンネルを作成する必要がないため、追加の処理遅延も発生しません。

さらに、VNet ピアリングではトラフィックが Azure 内部ネットワークに限定されるため、セキュリティも高くなります。

このように、**低遅延・高パフォーマンス・高セキュリティ**を同時に実現できるのが Global VNet Peering です。

## 6. 他の選択肢が誤りな理由

### B プライベートエンドポイントとの VNet ピアリング

Private Endpoint は Azure サービス（Storage、SQL など）へのプライベートアクセスを提供する機能です。  
VNet 同士を接続する用途ではありません。

### C VPN ゲートウェイ（VNet-to-VNet）

VPN ゲートウェイは IPsec トンネルを使用します。  
安全ですが暗号化処理が必要になるため、VNet Peering より遅延が大きくなります。

### D Azure Virtual WAN

Virtual WAN は大規模ネットワークアーキテクチャ向けのサービスです。  
単純な VNet 間接続には過剰な構成となります。

## 7. 最終回答

A. **グローバル VNet ピアリング**

## 8. まとめ

Azure の異なるリージョンの VNet を接続する場合、最も低遅延で安全な方法は **Global VNet Peering** です。

この機能は Azure バックボーンネットワークを使用するため、インターネットを経由せず高速通信が可能になります。

試験では **「VNet 間通信」「低遅延」「異なるリージョン」** といったキーワードが出た場合、**Global VNet Peering** が正解になるケースが非常に多いです。
